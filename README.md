 The Digital Sustainability Auditor

An interactive web tool that estimates digital carbon footprint and sustainability score using traffic, page weight, and infrastructure efficiency.

 Overview

The Digital Sustainability Auditor is a browser-based application designed to visualize the environmental impact of websites and digital platforms.

As global internet usage increases, data centers and network infrastructure consume large amounts of electricity. This project helps users understand how website traffic, page size, and server efficiency affect digital carbon emissions.

The goal is to promote awareness and encourage sustainable digital development.

 The Problem

Most developers optimize websites for speed and design — but rarely consider environmental impact.

Heavy pages, unoptimized assets, and inefficient infrastructure increase:

- Energy consumption
- Server load
- Carbon emissions

There is limited awareness about digital sustainability at the student and beginner level.



The Solution

This project allows users to:

- Enter monthly website visitors
- Input average page size (MB)
- Simulate server efficiency (1–10 scale)
- Generate:
  - Sustainability Score (0–100)
  - Estimated CO₂ output (kg/month)
  - Visual progress indicator
  - Environmental performance feedback

The tool converts technical metrics into a simple, understandable sustainability score.

How It Works

The system uses a simplified estimation model:

Data Transfer = Visitors × Page Size  
Estimated CO₂ = Data Transfer × Emission Factor  
Sustainability Score = 100 − (CO₂ ÷ Efficiency)

Higher traffic and larger page sizes increase emissions.  
Higher infrastructure efficiency improves sustainability score.
